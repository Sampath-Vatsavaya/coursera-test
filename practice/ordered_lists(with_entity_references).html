Steps to write and run a java program

Ordered list:
  1. Open Notepad
  2. Write your program in the notepad editor
  3. Save the file with <mainclassname>.java extension
  4. Compile and run the program
    1. Open Command Prompt and nabigate to the file
    2. Run "javac <filename>.java" command to compile
    3. Run "java <filename>" command to run the program
  5. Happy Programming
